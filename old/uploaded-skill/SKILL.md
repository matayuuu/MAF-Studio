# Support First Response Skill

## Purpose

このスキルは、社内ユーザーからのIT問い合わせに対して、
ヘルプデスク担当者が送る一次回答文を作成するためのものです。

このスキルを使うことで、回答のトーンと品質を揃えつつ、
必要な確認事項を漏れなく案内できます。

## When to use this skill

以下のような場合に使ってください。

- パスワードリセット依頼への返信
- VPN接続できない問い合わせへの返信
- ソフトウェアインストール申請への返信
- PC不具合の一次切り分け案内
- 問い合わせを適切な窓口に案内する返信

## When not to use this skill

以下の場合は、このスキルをそのまま使わないでください。

- 障害原因がすでに特定されており、詳細な技術回答が必要な場合
- セキュリティインシデントの疑いがある場合
- 人事・法務・経理などIT以外の問い合わせの場合
- 回答に社外秘情報や個人情報を含める必要がある場合

## Instructions

問い合わせ内容を読み取り、以下の方針で一次回答文を作成してください。

1. まず問い合わせへのお礼を短く述べる
2. 問い合わせ内容を1文で要約して認識合わせをする
3. すぐ案内できる対処法があれば簡潔に示す
4. 情報不足で判断できない場合は、必要な確認事項を箇条書きで示す
5. 別窓口対応が適切なら、その理由を簡潔に述べて案内する
6. 最後は丁寧に締める

## Response rules

- 日本語で回答する
- トーンは丁寧・簡潔・事務的
- 回答は長くしすぎない
- 不確かなことを断定しない
- 必要な追加確認事項は箇条書きにする
- ユーザーを責める表現は使わない
- 個人情報や機密情報の送付を安易に求めない
- パスワードそのものの送付は依頼しない
- セキュリティ事故の可能性がある場合は、通常回答ではなく緊急連絡先へ誘導する

## Standard troubleshooting prompts

問い合わせ種別ごとに、必要に応じて以下を確認してください。

### Password reset

- 対象システム名
- ログインID
- エラーメッセージの有無
- 多要素認証の利用有無

### VPN issue

- 利用端末（会社PC / 私物PC）
- 発生日時
- 自宅回線か外出先回線か
- エラーメッセージの内容
- 他の社内サービスへアクセス可能か

### Software installation request

- インストールしたいソフト名
- 利用目的
- 利用端末
- 管理者権限の要否
- 業務上の必要性

## Escalation rules

以下に該当する場合は、通常の一次回答ではなくエスカレーションしてください。

- 「不審なメールを開いた」「添付を実行した」などの記載がある
- マルウェア感染や情報漏えいが疑われる
- 複数ユーザーに同時多発している障害
- 役員・重要顧客対応中の緊急案件
- 業務停止に直結する重大障害

## Output format

以下の形式で回答文を作成してください。

件名:
<メール件名>

本文:
<一次回答文>

## Example

### Input

VPNに接続できません。今朝から急につながらなくなりました。

### Output

件名:
【一次回答】VPN接続不具合について

本文:
お問い合わせありがとうございます。

本日朝からVPNに接続できない事象が発生しているとのこと、承知しました。

まずは、PCの再起動後に再度VPN接続をお試しください。
あわせて、状況確認のため、以下をご共有ください。

- ご利用端末（会社PC / 私物PC）
- 表示されているエラーメッセージ
- ご自宅回線か外出先回線か
- 他の社内サービスへはアクセス可能か

内容を確認のうえ、必要に応じて次の対応をご案内いたします。
よろしくお願いいたします。
