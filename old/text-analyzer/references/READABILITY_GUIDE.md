# 読みやすさスコア ガイド

## スコア基準

| スコア | 評価 | 説明 |
|--------|------|------|
| 90-100 | 非常に読みやすい | 小学生でも理解可能 |
| 70-89 | 読みやすい | 一般的なビジネス文書に適切 |
| 50-69 | やや難しい | 専門知識が必要な場合あり |
| 30-49 | 難しい | 専門的な読者向け |
| 0-29 | 非常に難しい | 学術論文レベル |

## 計算要素

1. **平均文長** — 1文あたりの文字数（短いほど読みやすい）
2. **平均段落長** — 1段落あたりの文数（3-5文が理想）
3. **漢字含有率** — 全体に占める漢字の割合（30%前後が読みやすい）
4. **専門用語密度** — 一般的でない語の割合

## 改善ガイド

- 1文は **40-60文字** を目安に
- 1段落は **3-5文** にまとめる
- **箇条書き** を活用して情報を整理する
- 難しい漢字には **ふりがな** を振る
- 接続詞で **文のつながり** を明確にする
