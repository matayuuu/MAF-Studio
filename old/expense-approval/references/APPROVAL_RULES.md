# 経費承認ルール

## 承認フロー（金額別）

| 金額 | 承認者 | 備考 |
|------|--------|------|
| 5,000円未満 | 承認不要（自動承認） | 領収書の添付のみ |
| 5,000円〜30,000円 | 直属の上長 | |
| 30,001円〜100,000円 | 部長 | 事前申請が望ましい |
| 100,001円〜500,000円 | 本部長 | 事前承認 **必須** |
| 500,001円以上 | 役員（CFO） | 事前承認＋稟議書 |

## 特別承認が必要なケース

以下に該当する場合は、金額に関わらず **部長以上の承認** が必要です。

1. **接待交際費** — 1回あたり1名5,000円超の飲食を伴う場合
2. **慶弔見舞金** — 社内規程の金額を超える場合
3. **タクシー利用** — 終電後（23:00以降）以外の利用は理由書が必要
4. **海外出張** — すべて本部長の事前承認が必要
5. **IT機器購入** — 10,000円以上は情報システム部の事前確認が必要
6. **定期利用サービス** — SaaS等の月額契約は初回に部長承認＋IT部確認

## 添付書類ルール

| 経費種別 | 必要書類 |
|----------|----------|
| 交通費（電車・バス） | 経路と金額の記載（IC履歴可） |
| タクシー | 領収書（宛名：会社名）＋利用理由 |
| 飲食（1名5,000円以下） | 領収書 |
| 飲食（1名5,000円超） | 領収書＋参加者リスト＋目的 |
| 宿泊 | 領収書（宛名：会社名） |
| 備品購入 | 領収書＋品名明細 |
| IT機器 | 領収書＋情報システム部の承認メール |

## 締め日と支払い

- **申請締め日**: 毎月 **25日** （25日が休日の場合は前営業日）
- **支払日**: 翌月 **15日** に給与口座へ振込
- 締め日を過ぎた申請は翌月処理（3ヶ月以上前の経費は原則不可）

## よくある差し戻し理由

1. 領収書の宛名が個人名になっている（会社名が必要）
2. 飲食で参加者リストが添付されていない
3. タクシー利用の理由が未記入
4. 勘定科目の選択誤り（会議費 vs 交際費）
5. 消費税の内税・外税が不一致
