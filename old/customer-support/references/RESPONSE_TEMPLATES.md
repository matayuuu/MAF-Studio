## 初回応答

```

お問い合わせいただきありがとうございます。
{company_name} サポートチームの {agent_name} でございます。

少々お時間をいただけますでしょうか。

{sla_time} 以内に回答いたします。

何卒よろしくお願いいたします。
```


## 謝罪（サービス障害時）

```
{customer_name} 様

この度は {service_name} のサービス障害により
ご迷惑をおかけしましたこと、深くお詫び申し上げます。

■ 障害発生期間: {start_time} 〜 {end_time}
■ 影響範囲: {impact_scope}
■ 原因: {root_cause}
■ 恒久対策: {permanent_fix}

SLA に基づくサービスクレジットについては、
別途カスタマーサクセス担当よりご連絡いたします。

再発防止に全力で取り組んでまいります。
```

## 回答のトーン・マナー

- **敬語**: です・ます調で統一
- **感謝**: 冒頭で必ず問い合わせへの感謝を述べる
- **謝罪**: 不具合の場合は早期にお詫びを入れる
- **具体性**: 「確認します」だけでなく、いつまでに何をするかを明示する
- **禁句**: 「仕様です」「できません」→ 代替案を提示する
