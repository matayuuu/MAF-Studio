## よくある質問（FAQ）

### Q1: パスワードを忘れた場合は？
**A**: ログイン画面の「パスワードをお忘れの方」からリセットメールを送信できます。メールが届かない場合は迷惑メールフォルダを確認してください。それでも届かない場合はサポートまでご連絡ください（本人確認のため社員証の写真が必要です）。

### Q2: 二要素認証（2FA）の設定方法は？
**A**: 設定 → セキュリティ → 二要素認証 から設定できます。**推奨**: Microsoft Authenticator。対応アプリ: Google Authenticator, Authy。SMS認証は2026年6月に廃止予定です。

### Q3: データのエクスポート方法は？
**A**: ダッシュボード右上の「⋯」→「エクスポート」から CSV/Excel/PDF 形式で出力できます。大量データ（10万件超）の場合はバッチエクスポート機能をお使いください（設定 → データ管理 → バッチエクスポート）。

### Q4: API のレートリミットに引っかかる
**A**: 現在の制限: 100リクエスト/分。エンタープライズプランでは1,000リクエスト/分に引き上げ可能。バッチAPIの利用も検討してください（一度に最大100件処理可能）。

### Q5: 請求書の宛名を変更したい
**A**: カスタマーサクセス担当にご連絡ください。変更は翌月請求分から反映されます。過去の請求書の再発行は経理部で対応（2営業日かかります）。

## 頻出トラブルシューティング

### ログインできない
1. Caps Lock がオフになっているか確認
2. ブラウザのキャッシュ・Cookieをクリア
3. シークレットモードで試行
4. 上記で解決しない → アカウントロックの可能性（5回連続失敗でロック、30分後に自動解除）

### 動作が重い
1. ブラウザのタブを閉じてページを再読み込み
2. ブラウザ拡張機能を無効化して試行
3. 別のブラウザで試行
4. 上記で解決しない → ネットワーク速度を確認（推奨: 10Mbps以上）
