# 新入社員オンボーディングガイド

新入社員の入社から独り立ちまでの手続き・研修・環境構築を支援します。部門ごとの違いや、よくある漏れ・つまずきポイントを含めた実践的なガイダンスを提供します。

## Scripts

- `scripts/checklist.py` — 入社日・部門・雇用形態に応じた必要手続きチェックリストを生成します。
- `scripts/training_plan.py` — 職種・レベルに応じた研修スケジュールを作成します。

## References

- `references/PROCEDURES.md` — 入社手続きの詳細手順と担当部門
- `references/SYSTEM_ACCESS.md` — 各システムのアカウント申請方法と承認フロー
- `references/TRAINING_CURRICULUM.md` — 職種別の研修カリキュラムと必須受講科目
