# 自動車保険 提案ルール

## 年齢別推奨プラン
| 年齢 | 推奨プラン | 理由 |
|------|-----------|------|
| 18〜25歳 | P004 ヤング自動車保険 | 若者向け特別設計、万が一の備えを充実 |
| 26〜59歳 | P002 スタンダード / P003 プレミアム | ライフスタイルと予算に応じて選択 |
| 60歳以上 | P005 シニア安心自動車保険 | 高齢者リスクに特化した保障 |
| 予算重視 | P001 エコノミー | 最低限の賠償責任のみ |

## アップセル判断基準
- 年収500万円以上 → スタンダード以上を推奨
- 年収1,000万円以上 → プレミアムを推奨
- 新車・高額車 → 車両保険付きプランを推奨
- 家族持ち → 搭乗者傷害付きプランを推奨

## 禁止事項
- 年齢要件外のプランは提案しない（例: 26歳にヤング保険は提案不可）
- 既に同種の自動車保険がある場合は重複加入を避ける
