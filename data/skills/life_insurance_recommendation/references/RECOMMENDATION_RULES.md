# 生命保険 提案ルール

## ライフステージ別推奨プラン
| ライフステージ | 推奨プラン | 理由 |
|--------------|-----------|------|
| 独身・20代 | P009 がんトータルケア | 若年がんリスクへの備え、低保険料 |
| 既婚・子あり | P007 ファミリー定期 | 遺族保障として有効、住宅ローン連動 |
| 独身・30〜50代 | P006 安心終身 + P008 医療 | 死亡保障と医療を組み合わせ |
| 富裕層・経営者 | P006 安心終身（高額） | 解約返戻金による資産形成 |
| リタイア層 | P010 ゆとりライフ年金 | 老後収入の安定確保 |

## 保障ギャップ分析
- 年収の3〜5年分の死亡保障が目安
- 医療保険は入院日額5,000〜10,000円が標準
- がん保険は40歳以上に積極提案

## 禁止事項
- 年齢要件外のプランは提案しない
- 既に同種の保険がある場合、重複加入のリスクを説明した上で提案する
